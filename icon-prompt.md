# SketchyBar Toggle — Icon Generation Prompt

Use with Gemini (nano-banana / AI Studio) to generate a macOS app icon for the README.

## Prompt

```
A macOS app icon in the style of Michael Flarup / Apply Pixels: a polished,
3D-rendered squircle icon with rich depth, soft ambient lighting, and subtle
material textures.

The subject: two horizontal status bars — one on top sliding up and fading
out, one below sliding down into view — conveying a seamless toggle or swap
motion. The bars should have a minimal, clean aesthetic with small abstract
status indicators (dots, short lines) like a macOS menu bar.

Use a deep indigo-to-teal gradient background. The bars should be translucent
frosted glass with soft white glows at their edges. A small, elegant toggle
arrow or chevron between them hints at the swap action.

Style reference: the premium, tactile quality of Flarup's Juice icon set —
vibrant but refined, with subtle shadows, specular highlights, and the
characteristic macOS Big Sur squircle shape. The icon should read clearly at
16x16 and look stunning at 512x512.
```

## Settings

- Aspect ratio: 1:1
- Size: 2K or 4K
- Thinking level: high (for best quality)
- Output: PNG
